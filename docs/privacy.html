Privacy Policy

utsulog-clipflow  |  Last updated: 2026-04-19

1. Overview

utsulog-clipflow is a personal automation tool that uploads video clips to YouTube using the YouTube Data API v3. It runs locally on the operator's server and is not distributed to or used by the general public.

2. Data Collected

  • OAuth credentials – An access token and refresh token issued by Google are stored locally as token.pickle on the operator's server. These credentials are never transmitted to any third party.
  • Video metadata – Titles, descriptions, and tags provided by the operator are sent to the YouTube Data API solely for the purpose of uploading videos.

3. Data Not Collected

This application does not collect, store, or transmit any personal information about end users. It does not use analytics, advertising, or tracking services.

4. Use of YouTube Data API

utsulog-clipflow uses the YouTube Data API v3 with the https://www.googleapis.com/auth/youtube.upload scope to upload videos to the operator's own YouTube channel. No other YouTube data is accessed or stored.

Google's privacy policy applies to all data processed through the YouTube Data API: https://policies.google.com/privacy

You can revoke this application's access to your Google account at any time at: https://myaccount.google.com/permissions

5. Data Sharing

No data is shared with third parties. All credentials and video files remain on the operator's local server.

6. Data Retention

OAuth credentials (token.pickle) are retained until manually deleted by the operator or revoked via Google Account settings. Uploaded video files are deleted from local storage after a successful backup to the operator's cloud storage.

7. Contact

For questions about this privacy policy, please open an issue at:
